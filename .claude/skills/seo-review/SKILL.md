# 技能：内容页面 SEO 审查

使用此技能对网站的内容页面进行专注的 SEO 审查。目标是最大化目标受众的搜索可见性和用户体验。

## 何时使用

- 发布新内容页面之前
- 优化表现不佳的页面时
- 定期内容审计
- 重大内容更新后
- 针对新关键词时
- 改善页面排名和点击率

## 目标

每个内容页面应该针对相关搜索意图进行优化：
- 信息型搜索："[主题] 是什么"、"如何 [做某事]"
- 导航型搜索："[品牌/产品名称]"
- 交易型搜索："购买 [产品]"、"[服务] 价格"
- 比较型搜索："[A] vs [B]"、"最佳 [类别]"

---

## SEO 审查方法

遵循以下五个步骤进行完整的 SEO 审查。

### 步骤 1：识别目标关键词

在审查之前，识别页面的关键词集群和搜索意图。

#### 关键词集群模板

| 类型 | 模式 | 示例（产品评测） |
|------|---------|-------------------|
| **主要** | [主题/产品] | iPhone 15 评测 |
| **是什么** | [主题] 是什么 | JSON 是什么 |
| **如何做** | 如何 [动作] | 如何制作网站 |
| **为什么** | 为什么 [问题] | 为什么选择 React |
| **最佳** | 最佳 [类别] | 最佳 SEO 工具 |
| **对比** | [A] vs [B] | React vs Vue |
| **指南** | [主题] 指南/教程 | SEO 完整指南 |
| **问题** | [主题] 常见问题 | Next.js 常见问题 |

### 步骤 2：页面 SEO 审查

系统地检查所有页面 SEO 元素。

#### 标题标签（Title Tag）
- 长度：50-60 字符
- 包含主要关键词（尽量靠前）
- 具有吸引力和点击诱因
- 准确描述页面内容
- 避免关键词堆砌

#### 元描述（Meta Description）
- 长度：150-160 字符
- 包含主要关键词
- 使用行动号召（CTA）
- 提供明确的价值主张
- 吸引用户点击

#### URL 结构
- 简短且描述性
- 包含关键词
- 使用连字符分隔单词
- 避免特殊字符和参数
- 符合逻辑层次结构

#### 标题结构（H1-H6）
- 每页只有一个 H1
- H1 包含主要关键词
- 标题层次清晰（不跳级）
- H2/H3 包含相关关键词
- 标题具有描述性

#### 内容质量
- 原创且有价值
- 满足搜索意图
- 适当的内容长度（根据主题）
- 关键词自然分布
- 易读性好（短段落、列表）
- 包含多媒体元素（图片、视频）

#### 图片优化
- 使用描述性文件名
- 添加 alt 文本（包含关键词）
- 优化文件大小
- 使用现代格式（WebP）
- 响应式图片

### 步骤 3：精选摘要优化

验证内容结构是否能赢得精选摘要（Featured Snippet）。

#### 定义型摘要
- 在前 100 字内提供清晰定义
- 40-60 字的简洁答案
- 使用"[主题] 是..."格式

#### 列表型摘要
- 使用编号列表（步骤）
- 使用项目符号列表（要点）
- 3-8 个项目最佳
- 每项简洁明了

#### 表格型摘要
- 比较数据使用表格
- 清晰的表头
- 易于扫描的格式

#### 段落型摘要
- 直接回答问题
- 2-3 句话
- 包含关键信息

### 步骤 4：内部链接审查

检查内部链接结构。

#### 链接数量
- 每页 3-5 个相关内部链接
- 避免过度链接
- 链接到相关内容

#### 锚文本
- 使用描述性锚文本
- 包含目标页面关键词
- 避免"点击这里"等通用文本
- 自然融入内容

#### 链接位置
- 内容主体中的链接
- 相关内容推荐区域
- 面包屑导航
- 侧边栏相关链接

### 步骤 5：生成报告

使用报告模板记录发现。

#### 报告结构

**页面信息**
- URL
- 标题
- 目标关键词
- 当前排名（如有）

**SEO 评分**
- 标题标签：✅/⚠️/❌
- 元描述：✅/⚠️/❌
- URL 结构：✅/⚠️/❌
- 标题层次：✅/⚠️/❌
- 内容质量：✅/⚠️/❌
- 图片优化：✅/⚠️/❌
- 内部链接：✅/⚠️/❌
- 精选摘要潜力：✅/⚠️/❌

**发现的问题**
1. 问题描述
2. 影响程度（高/中/低）
3. 建议修复方案

**优先修复项**
1. 高优先级问题
2. 快速胜利（容易修复且影响大）
3. 长期优化建议

---

## 总结

审查内容页面的 SEO 时：

1. **识别目标关键词** — 确定主要关键词和搜索意图
2. **检查标题标签** — 50-60 字符，关键词靠前，吸引点击
3. **检查元描述** — 150-160 字符，行动号召，明确价值
4. **验证关键词位置** — 标题、描述、前 100 字、H2/H3
5. **审查内容结构** — 清晰标题、短段落、列表、多媒体
6. **优化精选摘要** — 简洁定义、编号步骤、表格对比
7. **检查内部链接** — 3-5 个链接、描述性锚文本、相关内容
8. **生成报告** — 记录评分、问题和优先修复项

**记住：** SEO 不是为了操纵搜索引擎——而是让需要内容的用户更容易找到它。每个优化都应该改善用户体验和内容价值。
