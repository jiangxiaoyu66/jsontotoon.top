FAQ

What is TOON serialization format? +

TOON is a lightweight serialization format optimized for LLMs that reduces token usage while maintaining data structure.

How much can I save by using TOON? +

Typically, TOON reduces token count by 30-50% compared to JSON, directly translating to API cost savings.

Can TOON handle complex JSON structures? +

Yes, TOON supports nested objects, arrays, and all standard JSON data types.

Is the conversion reversible? +

Yes, you can convert from JSON to TOON and back to JSON without any data loss.

Which LLMs work best with TOON? +

TOON works with all major LLMs including GPT-4, Claude, and other models that process text-based data.
